# ⚖️ BMI Calculator

A BMI calculator that takes height and weight as input, displays the BMI value with category (underweight/normal/overweight/obese), and supports metric and imperial units.

## Features
- 📏 Metric (kg/cm) and Imperial (lb/in) units
- ⚖️ Calculates BMI instantly
- 🏷️ Displays category (Underweight, Normal, Overweight, Obese)
- 🎨 Clean, responsive UI

## Usage
Open `index.html` in your browser.  
Enter weight and height, select units, and click **Calculate**.
