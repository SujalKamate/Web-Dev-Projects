# Alarm App

A basic alarm app executed with HTML, CSS, and JavaScript

## Features

- Select date and time.
- View a list of alarms that you can either delete or reset.
- Easy interface with no clutter.
- Get notified with a Pop-Up when the set time is reached.

## Run it

Open `index.html` in any modern browser.

## What it shows

- A simple card with app name and present time.
- A drop down box the selet date and an entry box to enter time.
- When the set time is reached, a pop-up is seen with a message.
