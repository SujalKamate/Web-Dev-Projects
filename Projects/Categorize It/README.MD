# 🗂️ Categorize It

## Description

Added Categorize It, an educational classification game where users are presented with related words and must identify the correct category they belong to. For example, **Apple, Banana, Mango → Fruits**.

## Features

* Word grouping challenges
* Category identification system
* Multiple topic categories
* Score tracking
* Instant answer validation
* Progressive difficulty levels
* Interactive learning experience

## Benefits

* Improves classification and reasoning skills
* Enhances vocabulary and general knowledge
* Strengthens pattern recognition abilities
* Makes learning fun and engaging

## Future Improvements

* Hint system
* Timed challenge mode
* Custom category creation
* Achievement badges
* Leaderboard system
* Daily category challenges

## Use Cases

* Vocabulary learning
* General knowledge practice
* Educational games
* Cognitive skill development

## Learning Outcome

Users develop stronger categorization, logical thinking, and word association skills by identifying common relationships between different words and concepts.
