# Git Workflow Playground

An interactive Git learning environment that visually demonstrates how commits, branches, merges, rebases, and resets affect repository history.

## Run it

Open `index.html` in any modern browser.

## Features

* Interactive commit graph visualization
* Create commits instantly
* Create and switch branches
* Merge branches visually
* Rebase simulation
* Reset and revert demonstrations
* Live repository state updates
* Pure HTML, CSS and JavaScript implementation

## Learning Objectives

The playground helps users understand:

* Commit history
* Branch creation
* HEAD movement
* Merge commits
* Fast-forward merges
* Rebasing workflows
* Reset vs revert

## Technical Highlights

* SVG-based commit graph rendering
* Dynamic graph layout algorithm
* State-driven Git simulation engine
* No external libraries
* Fully client-side execution

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
