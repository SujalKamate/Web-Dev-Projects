# Daily Wheels

Can't decide what to eat, watch or do today? Spin the wheel.

## Features

- Built-in presets: **Eat**, **Watch**, **Do something**. Switch with one click.
- Add or remove options on the fly; everything saves to `localStorage`.
- Hand-drawn SVG wheel that renders any number of slices in clean, warm
  earthtones — no gradients, no AI palettes.
- Smooth easing on the spin, plus a satisfying result card.
- Keyboard accessible. Respects `prefers-reduced-motion`.

## Run it

Open `index.html`. Use the preset chips to switch wheels, or edit the option
list to make your own.

## What it shows

- Generating SVG `<path>` slices with trigonometry.
- CSS transforms with custom cubic-bezier easing.
- A small reducer-style state pattern in vanilla JS.
